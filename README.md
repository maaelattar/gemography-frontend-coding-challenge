## Getting Started

Installation

`yarn`

To Start Project

`yarn start`

## Dependencies

- React
- React Router
- Prop-Types for type checking components props
- Axios as HTTP client
- React Query for managing data fetching operations
- Tailwind-CSS
  - Craco
  - Post-CSS
  - Auto-Prefixer
- Moment JS for date utils
- React Content Loader for easily creating placeholder loadings
